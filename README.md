# api-tester-ai-mcp

MCP server for API testing and validation tools.

## Tools

- **send_request** — Send HTTP requests with full control
- **validate_response** — Validate API responses against expectations
- **check_headers** — Analyze response headers for security
- **generate_curl** — Generate curl and fetch commands

## Usage

```bash
pip install mcp
python server.py
```

## Rate Limits

50 calls/day per tool (free tier).
